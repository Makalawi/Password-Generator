# 🔐 Password Generator

A simple Python application that generates secure and random passwords.

## Features

- Generate passwords of any length
- Includes uppercase and lowercase letters
- Includes numbers
- Includes special characters
- Easy to use from the command line

## Requirements

- Python 3.x

## How to Run

```bash
python password_generator.py
```

## Example

```
Password length: 12

Generated Password:
X#9a!P2m@Q7$
```
